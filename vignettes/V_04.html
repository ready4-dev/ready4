Find And Author Extensions To The Ready4 Framework

library("ready4")

Motivation

The ready4 package provides a “bare-bones” framework that is designed to be extended by other work. This vignette briefly summarises some examples of current extensions to the framework.

Code libraries

A standardised workflow for authoring code libraries (R packages) that extend the ready4 framework is supported by the ready4pack package. A table of the most up to date list of R package extensions to the ready4 framework can be generated using the make_pkg_extensions_tb function.

x <- make_pkg_extensions_tb()

A HTML version of the table can be printed using print_pkg_extensions.

print_pkg_extensions(x)
