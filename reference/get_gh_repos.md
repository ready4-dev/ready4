# Get github repositories

get_gh_repos() is a Get function that extracts data from an object.
Specifically, this function implements an algorithm to get github
repositories. The function returns Repositories (a character vector).

## Usage

``` r
get_gh_repos(org_1L_chr)
```

## Arguments

- org_1L_chr:

  Organisation (a character vector of length one)

## Value

Repositories (a character vector)
