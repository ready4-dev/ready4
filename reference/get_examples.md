# Get examples

get_examples() is a Get function that extracts data from an object.
Specifically, this function implements an algorithm to get examples. The
function returns Examples (a character vector).

## Usage

``` r
get_examples(vignettes_chr, term_1L_chr)
```

## Arguments

- vignettes_chr:

  Vignettes (a character vector)

- term_1L_chr:

  Term (a character vector of length one)

## Value

Examples (a character vector)
