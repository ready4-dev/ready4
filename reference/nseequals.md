# Non standard evaluation assignment function

Import of := function, implementing
[`rlang::dyn-dots()`](https://rlang.r-lib.org/reference/dyn-dots.html).
See rlang package documentation for detailed description.

## Value

Does not return a value. Used to implement non standard evaluation
within a dynamic dots context.
