# Update prototype function arguments list

update_pt_fn_args_ls() is an Update function that edits an object, while
preserving core object attributes. Specifically, this function
implements an algorithm to update prototype function arguments list. The
function returns Updated arguments (a list).

## Usage

``` r
update_pt_fn_args_ls(args_ls)
```

## Arguments

- args_ls:

  Arguments (a list)

## Value

Updated arguments (a list)
