# Transform character vector to numeric

transform_chr_to_num() is a Transform function that edits an object in
such a way that core object attributes - e.g. shape, dimensions,
elements, type - are altered. Specifically, this function implements an
algorithm to transform character vector to numeric. The function returns
Digits (an output object of multiple potential types).

## Usage

``` r
transform_chr_to_num(digits_chr)
```

## Arguments

- digits_chr:

  Digits (a character vector)

## Value

Digits (an output object of multiple potential types)
