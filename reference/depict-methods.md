# Depict (plot) features of model module data

depict() is a method that plots features of data contained in a model
module (or sub-module).

## Usage

``` r
depict(x, ...)
```

## Arguments

- x:

  A model module (an instance of a class that inherits from
  Ready4Module) or submodule (any S3 class instance)

- ...:

  Additional arguments

## Value

A ggplot, gg or other plot type class.
