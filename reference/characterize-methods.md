# Characterize model module data by generating (tabular) descriptive statistics

characterize() is a method that generates descriptive tabular summaries
about data contained in a model module.

## Usage

``` r
characterize(x, ...)
```

## Arguments

- x:

  A model module (an instance of a class that inherits from
  Ready4Module) or submodule (any S3 class instance)

- ...:

  Additional arguments

## Value

A data.frame, tibble or other table based class.
