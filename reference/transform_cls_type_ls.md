# Transform class type list

transform_cls_type_ls() is a Transform function that edits an object in
such a way that core object attributes - e.g. shape, dimensions,
elements, type - are altered. Specifically, this function implements an
algorithm to transform class type list. The function returns Tfmd class
type (a list).

## Usage

``` r
transform_cls_type_ls(cls_type_ls)
```

## Arguments

- cls_type_ls:

  Class type (a list)

## Value

Tfmd class type (a list)
