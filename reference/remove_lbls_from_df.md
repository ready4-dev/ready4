# Remove labels from dataframe

remove_lbls_from_df() is a Remove function that edits an object,
removing a specified element or elements. Specifically, this function
implements an algorithm to remove labels from dataframe. The function
returns Unlabelled data (a data.frame).

## Usage

``` r
remove_lbls_from_df(data_df)
```

## Arguments

- data_df:

  Data (a data.frame)

## Value

Unlabelled data (a data.frame)
